# 附錄 C　名相對照

正文裡，凡遇一個名相，都在它頭一次出現時順手用白話解過了；梵文拼音與變音符號，則一概收在這裡，免得擾了正文。

每一條，給出：**中文名相 — 羅馬拼音（梵）— 一句白話**。本書為大乘分冊，拼音以梵文為主；漢譯自有之詞、或屬後世判教所立之名目，不另注拼音，逕標（漢譯詞）或（判教名）。

---

## 一、位與名

- **等覺** — （漢譯詞）—— 等同、近似於覺而未即是覺；緊鄰妙覺之前的一位，本書第一部之所說。
- **妙覺** — （漢譯詞）—— 佛果之位名；《瓔珞》第四十二地、「無上地」；行盡之地。
- **無垢地** — （漢譯詞）—— 《瓔珞》為等覺位所立之別名；與《華嚴》十地第二「離垢地」（*vimalā*）同名異位，不可混判。
- **究竟位** — （漢譯詞）—— 唯識五位之第五、最末位；住無上正等菩提、出障圓明；與妙覺同指。
- **如來成正覺** — （漢譯詞）—— 《華嚴》語系所稱佛之究竟覺；無二無相、無行無止；與妙覺同指。
- **四十二賢聖** — （漢譯詞）—— 《瓔珞》自家的階位框架：十住·十行·十向·十地·無垢地·妙覺地。
- **六種性／六堅** — （漢譯詞）—— 《瓔珞》攝位之性：習種性·性種性·道種性·聖種性·等覺性·妙覺性。
- **五十二位** — （判教名）—— 後世（圭峰、天台攝十信入位）綜整之整數位次；本書沿用為方便指稱，義理處回經典語言。

## 二、補處與留

- **一生補處** — eka-jāti-pratibaddha（梵）—— 唯餘一生、下一生即補佛之處而成佛；等覺位最紮實的內涵。
- **一生所繫菩薩** — （漢譯詞）—— 《華嚴》語，與一生補處同義。
- **故留不證／留惑不證** — （漢譯詞）—— 本可取證而為成佛故按住不取；一生補處之定義性標記。
- **辟支佛** — pratyekabuddha（梵）—— 無佛之世自悟之聖者；一生補處若此刻取證所墮之果地。
- **十八不共法** — （漢譯詞）—— 唯佛獨具、不與二乘共之十八種功德；一生補處「乃至十八不共法」所指。
- **阿耨多羅三藐三菩提** — anuttarā samyak-saṃbodhi（梵）—— 無上正等正覺；究竟位所住者。

## 三、定與斷

- **金剛喻定** — vajropama-samādhi（梵）—— 喻如金剛之定；最後邊學三摩地，摧伏一切煩惱而不為所伏，一剎那斷盡最後無明。
- **金剛心／金剛智** — （漢譯詞）—— 等覺之利智；「無明盡相為金剛」。
- **薩婆若** — sarvajñā（梵，音譯）—— 一切智；「盡相無相為薩婆若」，佛智之稱。
- **無始無明住地** — （漢譯詞）—— 四住地之前更無法起之根本無明；最後所斷者；等覺知其終而不知其始。
- **元品無明／最後一品無明** — （判教名）—— 天台等後世判教指最後所斷無明之語；本書所依經論無此詞，經語作「無始無明住地」。
- **無間道／解脫道** — （漢譯詞）—— 正斷惑之道／斷已證滅之道；菩薩利根，剎那剎那能斷證，不必別起二道。
- **學人末後心** — （漢譯詞）—— 有學位的最後一念；金剛喻定之位置，道作為「道」的最後一念。
- **剎那** — kṣaṇa（梵，音譯）—— 極短之時量；「一剎那中三界頓斷」之時。
- **有學／無學** — （漢譯詞）—— 尚有可學可斷者／學盡無可復學者；金剛喻定生於有學之最末邊際。
- **二障** — kleśa-āvaraṇa / jñeya-āvaraṇa（梵）—— 煩惱障與所知障；金剛喻定現前時頓斷其最後俱生微細者。
- **轉依** — （漢譯詞）—— 所依之轉換；金剛喻定現前「頓證佛果圓滿轉依」之所稱。

## 四、其餘

- **法雲地** — dharmameghā（梵）—— 十地之第十；前一冊止筆之地，本冊起筆之前一站。
- **二乘** — （漢譯詞）—— 聲聞與辟支佛；其金剛喻定語境（成阿羅漢之末後心）與大乘（入如來地）有別，本書照錄不代判。
- **出障圓明** — （漢譯詞）—— 障已出盡之後的圓明；究竟位之相。
- **一切種智** — sarvākārajñatā（梵）—— 佛智之究竟；妙覺地「一切種智照達始終」所稱。
