# 過渡四 · 從十迴向到十地

第四部結束了。手上又多了兩樣東西。

一道工序：迴向。所積之善不順重力沉澱為私帳，而被指定撥付三處——向下歸眾生，向上趣菩提，向深順真如；三向一燈，獨無「歸己」一向。善不因迴向而失，反因迴向而轉大——「不壞迴向」的名字自己作了證。

一層不退：行不退。善既盡迴向、不留自果，利他之行便失去了可退守的私果立足點——破釜沉舟，無路可退，故行堅住。位不退守「是誰」，行不退守「在做」；三不退的前兩層，至此到手。

而三賢，也至此圓終。住位坐實了人，行位行出了事，迴向位交出了善。回頭看，從發心入門到此，走的全是「賢」的路——外凡入賢，賢中長成。往前看，是一道全書最重的分界：**賢與聖。**

十地，經中稱「十聖」。賢與聖之間隔著什麼？隔著一個字的差別：賢位於惑，是「伏」——壓伏煩惱，令不現行，根還在；聖位於惑，是「斷」——連根拔除，永不再起。伏如石頭壓草，搬開石頭草又生；斷如火燒種子，種子既焦，永不發芽。三賢走了三十位，惑伏而未斷；初地的門檻上，第一次真正動刀——見道，斷分別惑。讀過前一冊《四沙門果》的人，會在這裡認出一個熟悉的字眼：斷。聲聞道整條路就是斷結的減法；如今菩薩道走到十地，「斷」的語法回來了——同樣是斷，出口卻不同：聲聞斷盡即取證離場，菩薩斷而不取證、留惑潤生地往佛果走。同一把刀，兩本帳。這是第五部要細看的事。

第五部是全書最長的一部，四章，走十個地。第一章，初地歡喜地：見道——賢入聖的那一刀，以及為什麼這一地偏偏以「歡喜」為名。第二章，二地到七地：修道——分別惑斷了，俱生的習氣還在，六個地一地一地磨，障由粗入細。第三章，八地不動地：全書另一個重音所在——無生法忍，無功用任運，十自在，以及三不退的最後一個：念不退。第四章，九地十地：善慧地的法師之能，法雲地的灌頂受職——又一場灌頂，比十住那一場高了整整三十位——走到因位的最高處，停在等覺妙覺的門前。

三賢已畢，刀已在手。現在，跨過賢聖的分界，入地。
