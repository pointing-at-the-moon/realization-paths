# 過渡二 · 從十住到十行

第二部結束了。手上又多了兩樣東西。

一道門檻：入住是質的轉變，因為腳下托著心的東西換了——由會折舊的善根，換成入佛家、得佛姓的身分。位不退從此起算，但它鎖的只是「位」這一層：不墮二乘、不還凡夫；行與念的不退，還在後頭的路上。撐住這個不退的，是見地對根本兩難之不轉——聞有佛無佛、有法無法而心不動——不是覺受之盛。

一條養成弧線：十住是佛子在佛家裡長大的全程。名目自己透出王家血脈的線索——生貴、童真、王子、灌頂——從入家的初心，長到受職的灌頂，成就十智，連法王子也測不到他的境界。而這圓成仍是因位：加冕是開端，不是終局。

正是這個「開端」，把下一段路逼了出來。

受了職的人，不能一直留在家裡。灌頂住的十智——震動世界、照耀世界、調伏眾生——沒有一樣是關起門來受用的；它們全是**做事**的能。王受了冕，要臨朝；佛子受了職，要出門。出門去哪裡？去眾生那裡。做什麼？利他。怎麼做？六度——布施、持戒、忍辱、精進、禪定、般若。十住養的是這個人，十行用的是這個人：第三部要走的「十行」，正是把佛家裡長成的本事，拿到眾生堆裡去**行**出來的十步——從第一個「歡喜行」，到第十個「真實行」。

請留意這個轉向的份量。前兩部的重心，一直在行者自己身上：信穩不穩、位退不退、見地站沒站定——都是「我」的事。十行把鏡頭轉了過去：行的對象是眾生，行的內容是六度，行的場所是世間。大乘常說「自利利他」，到十行，利他第一次成了正課，不再是順帶。而這一轉，也把一個新的問題帶了進來：在家裡，見地站定就算穩；到了眾生堆裡，事情會不會把人沖走？布施遇到貪吝反彈、忍辱遇到橫逆相加，行得歡喜還是行得勉強？「行」這一層的退與不退——行不退——又要到哪裡才算數？第三部只有一章，但這一章要把「行位」的性格說清楚：為什麼十行的第一行偏偏叫「歡喜行」，第十行又憑什麼擔得起「真實」二字。

家已成，職已受。現在，開門，出去。
