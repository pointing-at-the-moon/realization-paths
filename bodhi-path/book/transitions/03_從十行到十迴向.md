# 過渡三 · 從十行到十迴向

第三部結束了。手上又多了兩樣東西。

一個轉向：行位把正課從「我」換成了眾生。前六行一一就是六度在行——惠施、淨戒、忍法、精進、正念，乃至住無所得——而每一行的箭頭都標著同一句話：但為饒益一切眾生。六度從菜單上的名目，成了端上桌的菜；利他不待修成而後行，踐行的當下就是利他。

一隻眼：五度離般若如盲，得般若將導，才名為有眼、才配「到彼岸」之名。福要智導；行善之際以無所得照之，世間的好事方升為波羅蜜之行。

但行得越密集，一個新問題越是按不住：積下來的這些善，往哪裡放？

這不是一個閒問題。善行會積成善果，這是因果的常理；而果，是會把人留住的。布施積福，福熟則感人天的安樂果報——修得越好，下一程的「好日子」越穩。可是菩薩道的目的，第一部就立了：不住。久應得涅槃尚且不取證，何況人天福報？於是行位之後，立刻逼出一道工序：所積的善，必須**轉向**——不讓它沉澱為自己的受用，而把它整批調撥出去。這道工序，聖典叫「迴向」。迴者，迴轉；向者，朝向——把善根的去向，從默認的「歸自己」，改判為別有所歸。

歸到哪裡？三賢的第三段「十迴向」給出三個受方：迴向眾生，迴向菩提，迴向實際。善歸眾生，是利他之向；善歸菩提，是成佛之向；善歸實際，是空慧之向——三個受方怎麼能同時成立，正是第四部要拆解的事。先看一眼十迴向的第一個名目，它幾乎是一道謎題：「救護一切眾生**離眾生相**迴向」。救護一切眾生，卻又離眾生相——既要救人，又不見有人可救？這個看似自相矛盾的名字，恰好把第三部末尾那隻「無所得」的眼，與迴向的功課接在了一起：迴向給眾生，而不取眾生之相。般若的眼，從行位一路看進迴向位。

還有一條線索要帶著走。三不退的第二個——行不退——傳統判教繫在迴向位前後。位不退鎖的是身分；行不退鎖的是什麼？利他之行行到什麼地步，才不再退轉為自了之行？十迴向是三賢的最後一段，走完它，前面就是十地——賢與聖的分界。迴向，是賢位的收官。

善已積在手上。現在，去學怎麼把它交出去。
