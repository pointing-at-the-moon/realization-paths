# 第一部 · 十信 · 先把門找對

這本書要走的，是菩薩道上的五十位：十信、十住、十行、十迴向、十地——從最初的一念信，走到法雲地的灌頂受職，停在等覺妙覺的門前。路極長，經說要三大阿僧祇劫；而起步之前，有一件事必須先辦妥：把這條路的門找對。

流傳很廣的一幅圖，把成佛之路畫成一道單軌的長梯：先修聲聞四果，成了阿羅漢，再「回小向大」接上菩薩道。照這幅圖，菩薩道的門開在阿羅漢果的樓頂。但聖典安門的位置不在那裡。菩薩道的門，開在發菩提心的那一念——發心即入，與四果的有無不相干。

第一部兩章，辦的就是這件事。第一章，把門找對：看《瑜伽師地論》《大智度論》《華嚴經》如何把「入大乘」乾淨俐落地繫在發心上，弄清楚菩薩道為什麼不是阿羅漢果之上加蓋的二樓，而是自發心起就分途的另一條路。第二章，看門內的第一段路：十信。門雖一念可入，門內這頭一段卻是全程最不穩的一段——信還會退。把「入門」與「站穩」分清楚，後面四十位的走法才不會錯估。

門要先找對，路才走得起。我們從第一章開始。
