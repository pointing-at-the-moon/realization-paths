# 第四部 · 十迴向 · 善的去向

行得越密集，積得越厚；積得越厚，一個問題越按不住：這些善，往哪裡放？

善根有它自然的去處——記在自己名下，沉澱成一筆待領的福。可是菩薩道的目的，開卷就立了：不住。連涅槃都不取證的人，豈能讓善根沉澱成自家的福帳？三賢的最後一段「十迴向」，辦的正是這道工序：把所積之善迴轉其向——不歸自己，而投向三處：眾生、菩提、實際。

第四部只有一章。一章要辦三件事：看「迴向」二字怎麼把善根從私帳上撥出去；看三個去向——向下的眾生、向上的菩提、向深的真如——如何是同一盞燈的三個及處；看為什麼善一旦盡數迴向、不留自果，菩薩的「行」反而再無退路可退——行不退，就立在這裡。

賢位的收官之筆。學著把善交出去。
