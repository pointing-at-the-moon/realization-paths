# 附錄 C　名相對照

正文裡，凡遇一個名相，都在它頭一次出現時順手用白話解過了；梵文拼音與變音符號，則一概收在這裡，免得擾了正文。

每一條，給出：**中文名相 — 羅馬拼音（梵）— 一句白話**。本書為大乘分冊，拼音以梵文為主；漢譯自有之詞、或屬後世判教所立之名目，不另注拼音，逕標（漢譯詞）或（判教名）。

---

## 一、門與乘

- **菩提心／發菩提心** — bodhicitta（梵）—— 為度一切眾生而求無上菩提之心；菩薩道唯一的入門，一念可入。
- **菩薩** — bodhisattva（梵）—— 發菩提心、行菩薩道之人；自發心即預菩薩數。
- **大乘** — mahāyāna（梵）—— 以成佛度生為目的之乘；與聲聞、辟支佛二乘為平行別途。
- **聲聞** — śrāvaka（梵）—— 聞佛聲教、修四沙門果而取證之行者；其道詳前一冊《四沙門果》。
- **辟支佛／獨覺** — pratyekabuddha（梵）—— 無佛之世自悟而不說法度眾之聖者；與聲聞合稱二乘。
- **回小向大** — （漢譯詞）—— 由二乘轉趣大乘之說；本書第一章所拆「先小後大」之圖即依附此語而生的誤讀。
- **種姓** — gotra（梵）—— 堪任成佛之根性；本性住與習所成二種。菩薩自有種姓，非由果位升級而成。
- **一切智** — sarvajñatā（梵）—— 佛之無所不徹之智；發心所求之的。
- **不取證／不住涅槃** — （漢譯詞）—— 到了該取證之處而不取；菩薩道之性格所在，與聲聞之取證相對。
- **二資糧** — puṇya-jñāna-saṃbhāra（梵）—— 福德與智慧兩種資糧；積累之路所積者。
- **三大阿僧祇劫** — asaṃkhyeya-kalpa（梵，阿僧祇）—— 三段無量數之劫；菩薩道全程之長。
- **波羅蜜** — pāramitā（梵）—— 義為「到彼岸」；六度得般若導，方成此名。

## 二、五十位與不退

- **十信** — （部名沿用 śraddhā，信）—— 五十位之首十位，住前外凡；信會退之潮間帶。
- **十住** — （部名沿用 sthāna，住）—— 第二個十位，入佛家為佛子；位不退起算。
- **十行** — （部名沿用 caryā，行）—— 第三個十位，出而利他、六度踐行。
- **十迴向** — （部名沿用 pariṇāmanā，迴向）—— 第四個十位，迴轉所積之善；三賢之終。
- **十地** — daśabhūmi（梵）—— 第五個十位，十聖；斷惑歷程之所在。
- **外凡／三賢／十聖** — （漢譯詞）—— 階位三段：十信為外凡，住·行·迴向三十位為三賢，十地為十聖。
- **位不退／行不退／念不退** — （判教名）—— 三不退：位不退（十住起，不墮二乘凡夫）、行不退（十迴向，利他之行不退）、念不退（八地，心意識行不現行）。「三不退」之專名係後世判教；經文作「上進分」「終不退轉」「阿鞞跋致」等。
- **阿鞞跋致** — avaivartika（梵）—— 不退轉；經論明文中「不退」之本名。
- **退分善根／上進分善根** — （漢譯詞）—— 《瓔珞》之對舉：信位屬退分（可退墮），發心住起屬上進分（上進不退）。
- **信想菩薩／假名菩薩／名字菩薩** — （漢譯詞）—— 住前依信假立之菩薩三名；身分未坐實之如實稱。
- **佛子** — （漢譯詞）—— 入三世諸佛家、得佛家之姓者；十住以上之實稱。
- **灌頂** — abhiṣeka（梵）—— 印度立太子即王位之受職禮，以水灌頂；十住之末與十地之末各有一場。
- **法王子** — （漢譯詞）—— 法王（佛）之子；十住第九位之名，亦九地菩薩之通稱。
- **墮在佛數** — （漢譯詞）—— 列入佛之數；十地受職之相，而猶非究竟佛。

## 三、六度與迴向

- **六波羅蜜／六度** — ṣaṭ-pāramitā（梵）—— 布施、持戒、忍辱、精進、禪定、般若；菩薩道之行體。
- **檀** — dāna（梵）—— 布施。十行之歡喜行即其行用。
- **尸羅** — śīla（梵）—— 持戒。饒益行即其行用。
- **羼提** — kṣānti（梵）—— 忍辱。無違逆行即其行用。
- **毘梨耶** — vīrya（梵）—— 精進。無屈橈行即其行用。
- **禪** — dhyāna（梵）—— 禪定。離癡亂行（成就正念）即其行用。
- **般若** — prajñā（梵）—— 智慧。五度之導；離之則五度如盲。
- **福智二分** — （漢譯詞）—— 六度之經緯：施戒忍屬福分，般若屬智分，精進禪定通二分。
- **無所得** — （漢譯詞）—— 般若空慧之行相；不取能所之相。善現行「住無所得」即配般若之據。
- **迴向** — pariṇāmanā（梵）—— 迴轉所向；把善根從沉澱自果改投他處之一轉。
- **三處迴向** — （教相綱攝名）—— 迴向眾生、迴向菩提、迴向實際；後世教相對十迴向去向之綜攝，三向各有經據。
- **真如** — tathatā（梵）—— 諸法如實之性；第八真如相迴向之所順，亦見道之所觀。
- **善知識** — kalyāṇa-mitra（梵）—— 能護持行人不退墮之良師善友；信位去向之所繫。

## 四、十地與斷惑

- **歡喜地** — pramuditā（梵）—— 初地。初證聖處多生歡喜；玄奘譯「極喜地」。
- **離垢地** — vimalā（梵）—— 二地。離犯戒煩惱之垢，淨戒具足。
- **發光地** — prabhākarī（梵）—— 三地。聞思修照法顯現；論名義作「明地」。
- **焰慧地** — arciṣmatī（梵）—— 四地。智火能燒煩惱薪；論名義作「焰地」。
- **難勝地** — sudurjayā（梵）—— 五地。出世智方便善巧，能度難度。
- **現前地** — abhimukhī（梵）—— 六地。名列難勝、遠行之間；其名義釋本書未展。
- **遠行地** — dūraṃgamā（梵）—— 七地。善修無相行功用，究竟能過世間二乘出世間道。
- **不動地** — acalā（梵）—— 八地。無能壞無能動，捨功用而任運。
- **善慧地** — sādhumatī（梵）—— 九地。無礙力說法，成就利他行；力波羅蜜增上。
- **法雲地** — dharmameghā（梵）—— 十地。得大法身具足自在，如雲普覆；智波羅蜜增上；灌頂受職之地。
- **見道／修道** — darśana-mārga / bhāvanā-mārga（梵）—— 初地現觀真如之位／二地以上漸斷餘惑之位；地道架構之二時。
- **見縛** — （漢譯詞）—— 見道所斷之惑；依分別邪見而起、迷於理者，見真如則應念頓斷。
- **分別惑／俱生惑** — （唯識判教名）—— 後世唯識以「分別起」配見道所斷、「俱生起」配修道所斷之名；經論明文作「見縛」「修道煩惱」。
- **伏／斷** — （漢譯詞）—— 壓伏（石壓草，根在）與斷除（除根，不復生）；三賢伏惑，十聖斷惑。
- **十障** — （漢譯詞）—— 十地各對治一障，自麁入細：凡夫我相、邪行、忘、解法慢、身淨我慢、微煩惱習、細相習、於無相有行、不能善利益眾生、於諸法中不得自在。
- **無生法忍** — anutpattika-dharma-kṣānti（梵）—— 忍可諸法不生不滅、於實相智慧無能壞無能動；成就此忍即入八地。忍，認可安忍之義，非忍耐。
- **無功用** — anābhoga（梵）—— 不假作意用力；八地捨功用行、得無功用法之相。
- **任運** — （漢譯詞）—— 不假作意而自運轉；玄奘譯本標八地之語。
- **十自在** — （漢譯詞）—— 八地菩薩所得：命、心、財、業、生、願、解、如意、智、法（玄奘本譯字小異）；對治十種怖畏。係菩薩因位之自在，與諸佛十自在法分屬不同名目。
- **煩惱障／所知障** — kleśa-āvaraṇa / jñeya-āvaraṇa（梵）—— 二障：障解脫之煩惱、障一切智之微細無知；二乘種姓唯證前者淨，菩薩種姓兼證二淨。
- **微智障** — （漢譯詞）—— 十地之上、佛地之前所餘最後一重極細之智障；對治此障故佛地別說。
- **等覺／妙覺** — （判教名）—— 後世判教於十地與佛地間所立之最後二階；本書系另冊專說。

## 五、其餘

- **三十七道品** — （漢譯詞）—— 三十七種趣道之行法；菩薩具足觀知而不取證之所修。
- **留惑潤生** — （漢譯詞）—— 菩薩斷而不盡取證、留一分惑以資受生度眾之機制。
- **大寮** — （漢譯詞）—— 寺院之廚房；本書第三部之喻所借。
- **阿耨多羅三藐三菩提** — anuttarā samyak-saṃbodhi（梵）—— 無上正等正覺；發心所求之果。
- **法身** — dharmakāya（梵）—— 法之身；法雲地「得大法身具足自在」所稱。
- **一切種智** — sarvākārajñatā（梵）—— 佛智之究竟；真如相迴向「皆悉迴向一切種智」之所向。
