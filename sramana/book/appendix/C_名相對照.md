# 附錄 C　名相對照

正文裡，凡遇一個名相，都在它頭一次出現時順手用白話解過了；梵文、巴利文的拼音與變音符號，則一概收在這裡，免得擾了正文。

每一條，給出：**中文名相 — 羅馬拼音（巴/梵）— 一句白話**。標「巴」的是巴利文（早期阿含脈絡），標「梵」的是梵文（論藏脈絡）；有些詞兩者都給，有些是漢譯自有之詞，不另注拼音。

---

## 一、四果與向果

- **沙門** — samaṇa（巴）/śramaṇa（梵）—— 出家修行之人；「四沙門果」即此道上的四個果位。
- **須陀洹／預流／入流** — sotāpanna（巴）/srota-āpanna（梵）—— 第一果。斷前三結，入八聖道之流，不墮惡道，極七返而盡苦。
- **斯陀含／一來** — sakadāgāmin（巴）/sakṛdāgāmin（梵）—— 第二果。三結斷，貪恚癡薄（九品斷其六），至多一返人間。
- **阿那含／不還** — anāgāmin（巴/梵）—— 第三果。五下分結盡，不還欲界，上生淨居而般涅槃。
- **阿羅漢** — arahant（巴）/arhat（梵）—— 第四果。十結俱斷，漏盡。本書寫到其「向」為止。
- **向／果** — （漢譯詞）—— 同一斷結的兩個相：向是正斷之動相（正在拆），果是已斷而住之靜相（已拆定）。
- **四雙八輩／四雙八士** — （漢譯詞）—— 四向四果，八種聖者；經中「世尊弟子僧」之內涵。

## 二、結——量果的尺

- **結** — saṃyojana（巴/梵）—— 繫縛；把心和輪迴綁在一起的繩結。果位以「斷了哪幾條」為量。
- **身見** — sakkāya-diṭṭhi（巴）/satkāya-dṛṣṭi（梵）—— 把五蘊身心執為「我」的見。前三結之一；擋「誰在走」。
- **疑** — vicikicchā（巴）/vicikitsā（梵）—— 對佛法僧道的根本猶豫。前三結之一；擋「走不走」。
- **戒禁取** — sīlabbata-parāmāsa（巴）—— 執與解脫無關的禁戒儀軌為能淨之道。前三結之一；擋「走哪條」。
- **欲貪** — kāma-rāga（巴/梵）—— 對欲界欲樂之貪。五下分結之四。
- **瞋恚** — paṭigha（巴）/pratigha（梵）—— 對違逆之境的恚怒。五下分結之五。
- **五下分結** — orambhāgiya-saṃyojana（巴）—— 繫人於欲界（下分）的五條：身見、戒禁取、疑、欲貪、瞋恚；盡之則不還欲界。
- **色貪** — rūpa-rāga（巴）—— 對色界定樂之微貪。五上分結之一。
- **無色貪** — arūpa-rāga（巴）—— 對無色界定境之微貪。五上分結之二。
- **慢** — māna（巴/梵）—— 「我」之最後殘影，常以「我已證」的細念現身。五上分結之三。
- **掉舉** — uddhacca（巴）/auddhatya（梵）—— 極靜之心最後一絲未寂之動。五上分結之四。
- **無明** — avijjā（巴）/avidyā（梵）—— 諸結之根，墊在一切認知最底層的根本痴。五上分結之五，斷在最後。
- **五上分結** — uddhambhāgiya-saṃyojana（巴）—— 繫人於上二界（色·無色）的五條；其整列術語出論藏，內容見於阿含。
- **九品（修惑）** — （論藏分判）—— 欲界貪瞋癡分上上至下下九個品級；「薄」即九品已永斷其六、唯餘下品。

## 三、斷與果體

- **擇滅** — pratisaṃkhyā-nirodha（梵）—— 以簡擇之慧力，使某煩惱永不再起；那個「永不再起」本身。無為法，故不退。沙門果之體。
- **無常想** — anicca-saññā（巴）—— 觀一切（含定樂、含「我」）皆無常之想；能連根斷上界諸結的最細之慧。
- **慧解脫** — paññāvimutta（巴）—— 不得深定而以慧漏盡的阿羅漢；須深經中世尊親自印可的一類聖者。
- **俱解脫** — ubhatobhāgavimutta（巴）—— 兼得深定與漏盡、兩面俱解脫的阿羅漢。
- **漏** — āsava（巴）/āsrava（梵）—— 滲漏；煩惱之總名。三漏：欲漏、有漏、無明漏。
- **漏盡** — （漢譯詞）—— 三漏永盡，十結俱了；減法之完成相，亦即阿羅漢果成立之剎。

## 四、受生與界

- **欲界／色界／無色界** — （漢譯詞）—— 三界：欲樂所繫之界、微妙色身與定樂之界、無色之純定境界。五下分繫欲界，五上分繫上二界。
- **極七返** — （漢譯詞）—— 須陀洹受生之上限：人天各極多七生（總二十八有）；「極」訓最多，非定額。
- **一往來** — （漢譯詞）—— 斯陀含受生之上限：天上一往，人間一返，於此般涅槃。
- **中有** — antarābhava（梵）—— 死此生彼之間的過渡之身；其有無為部派之諍，本書依《俱舍》述義。
- **淨居（五淨居天）** — suddhāvāsa（巴）/śuddhāvāsa（梵）—— 色界第四靜慮最上五天：無煩、無熱、善現、善見、色究竟；唯聖者所居，不還者生處。
- **阿迦膩吒／色究竟** — Akaniṭṭha（巴）/Akaniṣṭha（梵）—— 五淨居之頂，色界之極；上流者（修慧）之終點。
- **般涅槃** — parinibbāna（巴）/parinirvāṇa（梵）—— 究竟入滅。
- **五種般涅槃** — （漢譯詞）—— 不還者趣涅槃的五種走法：中般（中有位即般）、生般（生已不久）、有行般（多加行）、無行般（任運）、上流（展轉上生方般）。同果而異程。
- **四不壞淨** — （漢譯詞）—— 入流者之四記號：於佛、法、僧不壞之淨信，與聖戒成就。
- **八聖道／八正道** — ariya aṭṭhaṅgika magga（巴）—— 正見乃至正定；「入流」所入之流，即此道。

## 五、其餘

- **四諦** — （漢譯詞）—— 苦、集、滅、道；漏盡智證明所依之如實知見。
- **三明** — （漢譯詞）—— 宿命、天眼、漏盡三種明；本書所引「漏盡智證明」為其第三。
- **自證定型句** — （漢譯詞）—— 「我生已盡，梵行已立，所作已作（辦），自知不受後有」；阿羅漢漏盡之自我印證，通篇為了結之語。
- **苦邊際** — （漢譯詞）—— 苦之相續到此封邊；自此後苦不相續。
