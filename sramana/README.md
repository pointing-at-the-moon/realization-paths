# Four Śrāmaṇa Fruitions
## 《四沙門果》

**Division 1 of 3 · [The Realization Paths](../README.md)** — 行 軸第 1 分區。

> 須陀洹・斯陀含・阿那含・阿羅漢 — the śrāvaka path's four fruitions, read as a measurable **subtraction**: each fruition defined by *which fetters are gone*, irreversibly.

- **Author**: 釋慧鏡 (Shi Huijing)
- **License**: [CC BY-NC-SA 4.0](../LICENSE)
- **Status**: ✅ **Complete** — 9 papers across 5 parts, bilingual (中文 / English).

### Core thesis

四沙門果不以「證受」量,而以「斷結」量——一道可量度的減法,量的是「少了什麼」。自前三結之斷(定向),至十結俱盡之漏盡(門檻),逐牆而拆,至無一可拆而竟。This division walks the four stations to the threshold of 阿羅漢果; the arahant's *mastery* (自在) belongs to the twin axis, `attainment-signatures/arhat`.
