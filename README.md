# The Realization Paths
## 行——從四沙門果到等覺妙覺的階位歷程

*The third gate of the [指月 / Pointing-at-the-Moon](https://github.com/pointing-at-the-moon) fourfold 信解行證 — the staged path of realization, from the four śrāmaṇa fruitions through the fifty-two bodhisattva stations to equal-and-sublime awakening.*

華嚴以**信、解、行、證**四字攝盡修道全程。八部曲立其**信**(般若見地),六行門明其**解**(行門方法);此系列走其**行**——把見與解放進**階位歷程**,一站一站走過去。

> 階位歷程 · walking the stations.

從聲聞的**四沙門果**起步,接菩薩的**五十二位**(十信、十住、十行、十迴向、十地),終於**等覺、妙覺**。聲聞道與菩薩道在此並列上升:同自下而上,śrāvaka → bodhisattva → Buddha。

### 與其他三軸的接縫 / Seams with the other axes

行(本軸)與證(`attainment-signatures`)是結構上的攣生兄弟,在兩端相接:

- **四沙門果之阿羅漢果** ──► 證軸之 **阿羅漢自在**(兩軸第一分區)
- **等覺妙覺之妙覺** ──► 證軸之 **如來自在**(兩軸末分區)

行講「走到」,證講「到達後的自在」。階位是路,自在是抵達的簽名。
